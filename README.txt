Programa Separador de fotos;

APENAS PARA PC COM SO WINDOWS

Programa realiza a separação de fotos em pastas de imagens que contém pessoas e imagens que não contém pessoas.
Você não precisa mais depender do google fotos para realizar essa separação pra ti :)
Funciona Totalmente OFFLINE.

//Para testes e debugger
Necessário instalar 
* JAVA JDK 8
* OpenCV
* JavaFX
* Eclipse (Opcional)

 Realizar correta configuração do JavaFX na IDE, caso necessário.
